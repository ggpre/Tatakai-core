## Overview

Tatakai is a modern anime streaming platform built with:
- React 18 + TypeScript
- Vite for fast development
- Supabase for backend services
- Tailwind CSS + shadcn/ui for beautiful UI
- HLS.js for video streaming

## Key Features

### For Users
- Stream anime with HLS video player
- Search and discover content
- Continue watching across devices
- Create watchlists and favorites
- Comment and rate anime
- Customizable themes

### For Admins
- User management (ban/unban, promote)
- Maintenance mode
- Broadcasting messages
- Analytics dashboard
- Comment moderation
